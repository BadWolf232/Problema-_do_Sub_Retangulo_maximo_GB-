
# Problema: Identificando a Região de Interesse de uma Imagem

O trabalho busca abordar um problema relacionado a arrays bidimencionais (matrizes) com valores positivos e negativos, no qual teriamos que encontrar o subretangulo com a maior soma dentros da matriz.

- ``Python``

- ``Numpy``

- ``Time``

## Instalação

Instale projeto dos retangulos de soma máxima

- Instale o python em sua máquina:
    - No windowns: [Acesse esse link](https://www.python.org/downloads/)
    - Linux: 
        - Para conferir a versão da linguagem:
            -Para versões mais recentes de sistemas GNU/Linux:

            ```bash
                $ which python3 
                /usr/bin/python3
            ```
            - Em Linux Ubuntu: 

            ```bash
                $python3 --version
            ```

            - Instalação em gerenciador de pacotes: 
                - Ubuntu/Debian
                    ```bash
                     sudo apt-get install python3
                    ```
                - RedHat e CentOS
                    ```bash
                        sudo yum install python3
                    ```

- Será necessário instalar a biblioteca: 
    - Numpy: 
        - No Linux/Windowns:
            ```bash
             pip install Numpy
            ``` 
    * A biblioteca Time já é integrada ao python
